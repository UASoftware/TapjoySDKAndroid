Tapjoy Android SDK

If you are using Tapjoy managed currency, see the EasyApp Publisher project for examples.
If you are using NON-Managed by Tapjoy currency, see the Easy App Multiple Currency project for examples.

Both of these sample projects use the same Tapjoy SDK, located in the TapjoyConnect folder.

For information about SDK integration, best practices and FAQs, please go to http://kc.tapjoy.com
